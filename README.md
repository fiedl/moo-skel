MooOS skeleton files
====================

mooOS files and directories


Install with
------------

    $ makepkg -sfic PKGBUILD

example:
    
     $ rm -r ~/abs/moo-skel && mkdir -p ~/abs/moo-skel && cd ~/abs/moo-skel && wget https://raw.github.com/idk/moo-skel/master/PKGBUILD && makepkg -sfi

Remove with
-----------

    # pacman -Rs moo-skel
